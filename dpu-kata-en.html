DPU Kata-Runtime Offloading and Performance Optimization

Background

Volcano Engine DPU team is dedicated to offloading virtualization to DPU (Data Processing Unit) to improve deployment density and reduce costs. Kata Container is an open-source secure container runtime at ByteDance. This project completed QEMU Kata Container virtualization offloading to DPU.

Main Work

  • Completed QEMU Kata Container virtualization offloading to DPU
  • Completed emulated device offloading to DPU
  • Optimized for high-density deployment scenarios
  • Optimized for high-concurrency startup scenarios

Results

  • Improved deployment density by 1x
  • 4 patents granted

Tech Stack

  • Kata Container
  • QEMU
  • DPU Virtualization
  • VFIO
