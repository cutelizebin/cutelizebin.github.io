项目

  • KVCache 存储

    目标：统一管理 DRAM/SSD 上的 KVCache，作为独立的组件 daemon，与推理框架（vLLM/sglang/内部框架）解耦，同时上报 KVCache metadata 给调度器做 KVCache-aware 调度。[详情]

  • KVCache 模拟器

    用于预估特定模型需要的 KVCache 容量，评估 KVCache 感知调度算法的有效性，节约测试占用卡时和人力。

  • DPU 上的 Kata-Runtime 卸载和性能优化

    完成 QEMU Kata Container 到 DPU 上的虚拟化卸载，完成模拟设备到 DPU 上的卸载工作，针对高密部署、高并发启动做优化，将部署密度提升 1x。[详情]

  • Titanium: 轻量级 ARM TrustZone VMM

    利用 ARMv8.4 SEL2 新特性，在 TrustZone 中实现轻量级 VMM，与华为海思合作，包含 C 和 Rust 版本。

  • 公有云 IOMMU 性能优化

    研究优化 SRIOV + VFIO 方案中 IOMMU PageTable NUMA 分配不均的问题。
